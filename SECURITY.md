# نکات امنیتی

- رمز مدیر، توکن‌ها و کلیدها را داخل GitHub Commit نکنید.
- بعد از اولین ورود، `SUDO_PASSWORD` را از Variables حذف کنید.
- احراز هویت دومرحله‌ای GitHub و Railway را فعال کنید.
- از رمز حداقل 20 کاراکتری و تصادفی استفاده کنید.
- قبل از هر Redeploy از فایل `/var/lib/marzban/db.sqlite3` نسخه پشتیبان بگیرید.
- این استقرار برای استفاده آزمایشی و کم‌مصرف طراحی شده است.
- Railway پورت خارجی TCP را خودش تعیین می‌کند؛ پورت خارجی الزاماً 443 نیست.
